Programmes 2022

À propros de Programmes-2022.fr

www.programmes-2022.fr est un site internet destiné à exposer en ligne de manière claire et équitable les programmes de l'ensemble des candidats à l'Élection présidentielle de 2022. Le code source de ce site, ainsi que son contenu sont ouverts et disponibles sur GitHub .

Hébergement

Ce site internet, édité à titre non professionnel est hébergé par l'entreprise GitHub B.V. siégant à l'adresse Vijzelstraat 68-72, 1017 HL Amsterdam (Pays-Bas) via son service Github Pages. Les coordonnées des contributeurs ont été transmises à l'entreprise assurant l'hébergement du site.

Données personnelles

Ce site internet ne collecte aucune donnée personnelle et n'effectue aucune mesure d'audience. GitHub peut être amené à stocker les adresses IP des visiteurs, plus d'informations sur le site de l'hébergeur
